# IR Helper App

Este projeto permite importar operações de ações, FIIs, BDRs, dividendos e aluguéis para calcular automaticamente o imposto de renda.

## Backend (FastAPI)
- Roda localmente na porta 8000
- Upload via endpoints:
  - /upload-trades
  - /upload-dividendos
  - /upload-alugueis
